## 🌟 About the Project

**JobStack** is a full-stack job portal designed to simplify the hiring process for both **recruiters** and **job seekers**. It provides a seamless platform where recruiters can post and manage job opportunities, while candidates can discover jobs, apply effortlessly, and track their applications—all through a clean, responsive, and user-friendly interface.

Built using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**, JobStack follows modern development practices with secure authentication, RESTful APIs, and a scalable architecture.

---

## ✨ Key Features

🔐 **Secure Authentication** – JWT-based login and registration

👥 **Role-Based Access** – Separate dashboards for Recruiters and Job Seekers

💼 **Job Management** – Create, edit, update, and delete job postings

🔍 **Smart Job Search** – Browse and search available opportunities

📄 **Easy Applications** – Apply for jobs and track application status

🏢 **Company Profiles** – Manage company information and job listings

📱 **Responsive Design** – Optimized for desktop, tablet, and mobile devices

⚡ **Fast REST APIs** – Efficient backend built with Express.js

🎯 **Modern UI/UX** – Clean, intuitive, and interactive interface
